Limpiar Datos de Ventas

Elimina todas las órdenes, transacciones y cierres de caja. NO elimina productos del menú ni meseros. Útil para entregar la aplicación como nueva o resetear datos de prueba.

  • Se eliminarán todas las órdenes (completadas, pendientes, canceladas)
  • Se eliminarán todas las transacciones
  • Se eliminarán todos los cierres de caja
  • Las mesas volverán a estado "Disponible"
  • Los productos del menú se mantendrán
  • Los meseros se mantendrán
